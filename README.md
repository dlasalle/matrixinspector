MatrixInspector
===============

A tool for viewing and manipulating sparse matrices.


![unzoomed](screenshots/full.png)
![zoomed](screenshots/zoom.png)


Build Dependencies 
------------------

- CMake
- C++ compiler support C++11
- wxWidgets 3 (development headers)


Runtime Dependencies
--------------------

- wxWidgets 3



Building
--------
```
./configure && make
```


Installation
------------
```
make install
```
